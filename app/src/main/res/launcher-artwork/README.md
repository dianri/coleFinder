# ColeFinder — arte del launcher

Los iconos activos son **Adaptive Icons** en XML:

- Fondo: `drawable/ic_launcher_background.xml` (degradado petróleo / teal).
- Primer plano: `drawable/ic_launcher_foreground.xml` (pin + tejado).
- Monocromo (temas Android 13+): `drawable/ic_launcher_monochrome.xml`.
- Enlaces: `mipmap-anydpi/ic_launcher.xml` y `ic_launcher_round.xml`.

Para sustituir por un PNG de alta calidad exportado desde Figma/Illustrator: en Android Studio, **File → New → Image Asset**, selecciona la imagen y regenera; o coloca aquí los PNG finales y actualiza los XML según la guía de Adaptive Icons.
